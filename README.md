# 项目简介

86-基于深度学习的手势手语检测识别系统，是一款集手势识别与手语翻译于一体的智能应用。系统核心功能模块包括基于yolov5的实时手势检测与基于深度神经网络的识别算法。通过自定义的数据集训练，系统可准确识别多种手势和手语动作。该系统配备用户友好的UI界面，采用Python编写，实现了高效的交互体验。技术特点在于其准确率高、实时性强，适用于无障碍交流、远程控制等多个领域。


## 核心功能
- 基于yolov5的实时手势检测
- 基于深度神经网络的识别算法
- 多种手势和手语动作识别
- 用户友好的UI界面
